## 📖 项目简介

**Yu_ToolBox** 是一个基于纯静态 HTML/CSS/JavaScript 构建的嵌入式开发工具导航站。项目将嵌入式开发中常用的高频工具集成在统一的界面中，帮助开发者快速定位并使用所需工具，提升开发效率。

目前包含的工具：
- 🎨 **图片取模 / 渲染工具** — 支持多种颜色格式互转、取模与数组渲染，含 Floyd 抖动算法
- 🔋 **电池功耗计算器** — 根据电池容量估算设备工作时长

> 更多工具持续开发中……

---

## 🚀 功能特性

- **分类工具导航** — 按功能分类展示工具卡片，一目了然
- **实时搜索** — 支持按工具名称、描述、分类关键词快速筛选
- **明暗主题切换** — 支持 Light / Dark 双主题，适配不同使用环境
- **响应式设计** — 完美适配桌面、平板、手机等设备
- **一键加入交流群** — 快速加入 QQ 技术交流群（453705020）
